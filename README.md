## Prerequisites & Requirements

Before running the Ultras-v3 suite, ensure you have the required classes, enhancements, and damage boosts unlocked on your account.

### Required Classes
You must own the following classes:
*   Verus DoomKnight
*   Lord of Order
*   StoneCrusher
*   King's Echo
*   ArchPaladin
*   Dragon of Time
*   ArchFiend

### Forge Enhancements
| Slot | Requirements |
| :--- | :--- |
| **Weapon** | Minimum of **Valiance**, **Lacerate**, or **Praxis** |
| **Helm** | All Helm Forges unlocked |
| **Cape** | All Cape Forges unlocked |

### Damage Boosts
To ensure the bot meets the strict DPS requirements for burst phases, you must equip boost gear:
*   **Weapon:** Minimum **40% damage** boost to all monsters.
*   **Armor/Pet:** Minimum **30% damage** boost to all monsters.
*   *Example Setup:* Hollowborn Reaper's Scythe + ArchFiend DoomLord (AFDL).

## How to Use Ultras-v3

You can automate your entire routine or run specific bosses manually depending on your needs.

### Option 1: Automated Daily/Weekly Run (DoAllUltras.cs)
*   **Daily First:** Running this script executes all daily Ultras first, followed by the weekly Ultras. 
*   **Daily Routine:** You can safely run DoAllUltras.cs every day to automate your daily Ultra clears.
*   **Beta Note:** Automated progression through all 11 Ultras back-to-back is still undergoing thorough testing. 

### Option 2: Manual/Individual Runs
If you encounter any logic loops, disconnecting issues, or bugs while running the full automation script, please navigate to the IndividualUltras folder and run the specific boss script you need manually.
